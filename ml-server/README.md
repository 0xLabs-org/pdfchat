# ML Server

## To Start Locally

- `cd ml-server`
- `.\venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Linux/Mac)
- `pip install -r requirements.txt`
- `python app.py`

## Contribute

- Follow contribution guidelines in the [CONTRIBUTING.md](CONTRIBUTING.md) file.
